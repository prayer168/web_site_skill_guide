一、CLI 主要命令

啟動會話、管道輸入、恢復對話與管理更新

§ 01
命令 說明 範例
claude 啟動互動式會話 claude
claude "query" 帶初始提示啟動會話 claude "explain this project"
claude -p "query" 單次查詢後退出（非互動模式） claude -p "explain this function"
cat file │ claude -p "query" 處理管道輸入內容 cat logs.txt │ claude -p "explain"
claude -c 繼續最近一次的對話 claude -c
claude -c -p "query" 以 SDK 模式繼續對話 claude -c -p "Check for errors"
claude -r "<session>" "query" 以 ID 或名稱恢復特定會話 claude -r "auth-refactor" "Finish PR"
claude update 更新到最新版本 claude update
claude install [version] 安裝或重新安裝指定版本 claude install stable
claude auth login 登入 Anthropic 帳號 claude auth login --console
claude auth logout 登出帳號 claude auth logout
claude auth status 顯示認證狀態（JSON 格式） claude auth status --text
claude agents 列出所有設定的子代理 claude agents
claude auto-mode defaults 輸出自動模式預設規則 claude auto-mode defaults > rules.json
claude mcp 設定 MCP 伺服器 claude mcp add github https://mcp.github.com
claude plugin 管理 Claude Code 外掛 claude plugin install code-review@official
claude project purge [path] 刪除專案的所有本地狀態 claude project purge ~/work/repo --dry-run
claude remote-control 啟動遠端控制伺服器 claude remote-control --name "My Project"
claude setup-token 產生長效 OAuth token claude setup-token
claude ultrareview [target] 非互動式執行 ultrareview claude ultrareview 1234 --json

二、CLI 旗標（Flags）

自訂 Claude Code 行為的所有旗標，依功能分六類

§ 02

2.1 會話控制

Session
旗標說明
-c / --continue繼續最近的對話
-r / --resume恢復指定會話（ID 或名稱）
-n / --name設定會話顯示名稱
--fork-session恢復時建立新 session ID
--from-pr恢復與 PR 關聯的會話
--session-id使用特定 UUID 作為 session ID

2.2 輸出與除錯

Output / Debug
旗標說明
-p / --print非互動模式，直接輸出結果
--output-format指定輸出格式：text、json、stream-json
--max-turns限制最大對話輪數（非互動模式）
--max-budget-usd限制最大花費金額
--verbose顯示詳細日誌
--debug啟用除錯模式（可指定分類）
--debug-file <path>將除錯日誌寫入指定檔案

2.3 模型與效能

Model / Performance
旗標說明
--model指定模型（sonnet、opus 或完整名稱）
--effort設定努力程度：low / medium / high / xhigh / max
--fallback-model模型過載時自動備援（非互動模式）
--betas包含 Beta 標頭（API key 使用者）

2.4 系統提示詞

System Prompt
旗標說明
--system-prompt取代整個系統提示詞
--system-prompt-file從檔案載入替換系統提示詞
--append-system-prompt附加到預設系統提示詞末尾
--append-system-prompt-file從檔案附加到系統提示詞

2.5 工具與權限

Tools / Permissions
旗標說明
--permission-mode設定權限模式（default / plan / auto / bypassPermissions 等）
--allowedTools不需確認即可執行的工具
--disallowedTools完全禁用的工具
--tools限制可用的內建工具
--dangerously-skip-permissions跳過所有權限提示（危險！）
--allow-dangerously-skip-permissions將 bypassPermissions 加入 Shift+Tab 循環
⚠️
注意：--dangerously-skip-permissions 會跳過所有安全確認，請僅在完全受控的環境中使用。

2.6 其他實用旗標

Misc
旗標說明
--add-dir新增額外工作目錄
--mcp-config從 JSON 載入 MCP 伺服器設定
--worktree / -w在獨立 git worktree 中執行
--bare最精簡模式，跳過自動探索
--chrome啟用 Chrome 瀏覽器整合
--ide啟動時自動連接 IDE
--remote在 claude.ai 建立新 web 會話
--remote-control / --rc啟動互動會話並啟用遠端控制
--agent指定本次會話使用的 agent
--tmux為 worktree 建立 tmux 會話
-v / --version顯示版本號

三、快速安裝指南

五個步驟，從安裝到啟動

§ 03
01
安裝 Claude Code
npm install -g @anthropic-ai/claude-code
02
登入帳號
claude auth login
03
確認登入狀態
claude auth status --text
04
啟動互動會話
claude
05
更新到最新版本
claude update
💡
提示：輸入錯誤的子命令時，Claude Code 會自動建議最接近的正確命令。
