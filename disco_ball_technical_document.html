1. 製作目標

本範例要製作一顆懸掛在畫面中央的 DISCO 鏡面球。啟動後，球體會旋轉， 多道彩色光束會從球的中心向四面八方放射；關閉後則停止動畫並隱藏光束。

HTML 建立球體、光束、按鈕及狀態文字等網頁元素。
CSS 負責鏡面格紋、光線、顏色、動畫及響應式版面。
JavaScript 接收按鈕操作，開啟或關閉旋轉與發光效果。

2. 核心原理

2.1 所有光線必須共用同一個起點

光束看起來像是從 DISCO 球射出，最重要的條件是： 每一道光線都放在球體中心，並將旋轉原點設定為光線的左端。 

.ray {
  position: absolute;
  left: 0;
  top: 0;
  transform-origin: 0 50%;
}

2.2 使用不同角度向外排列

同一條水平光線可以透過 rotate() 旋轉成不同方向。 

.ray1 { transform: rotate(0deg); }
.ray2 { transform: rotate(30deg); }
.ray3 { transform: rotate(60deg); }
光線並不是由球體本身真正產生，而是將光束元素放在球體後方， 讓球體遮住光束起點，因此視覺上看起來像從球內射出。

3. HTML 網頁結構

主要結構可分成四個部分：

<div class="disco-system">

  <div id="rays" class="rays">
    <div class="ray ray1"></div>
    <div class="ray ray2"></div>
  </div>

  <div class="wire"></div>
  <div id="discoBall" class="disco-ball"></div>
  <div id="lightCore" class="light-core"></div>

</div>
元素 功能
disco-system 整個 DISCO 球與光線效果的共同定位中心。
rays 包住全部光線，整體旋轉時只需旋轉這個容器。
disco-ball 顯示鏡面球的格紋、立體陰影與旋轉。
light-core 中央白色光源，強化光線從球內射出的效果。

4. 製作 DISCO 球外觀

使用兩層重複線性漸層製作橫向與直向格線，再加入放射漸層， 讓圓形呈現金屬鏡面球的明暗變化。 

.disco-ball {
  width: 260px;
  height: 260px;
  border-radius: 50%;
  overflow: hidden;

  background:
    repeating-linear-gradient(
      0deg,
      transparent 0 19px,
      rgba(20, 20, 20, 0.6) 19px 22px
    ),
    repeating-linear-gradient(
      90deg,
      transparent 0 19px,
      rgba(20, 20, 20, 0.6) 19px 22px
    ),
    radial-gradient(
      circle at 32% 25%,
      #ffffff 0%,
      #eeeeee 12%,
      #a8a8a8 40%,
      #555555 72%,
      #202020 100%
    );
}
常見誤解： DISCO 球本身通常是反射外部燈光，不是主動發光的燈具。 本教學為了呈現動畫效果，採用「視覺模擬」方式讓光線看似由球體射出。

5. 製作放射光線

每一道光束都是一個細長的長方形，使用透明漸層讓末端逐漸消失。 

.ray {
  position: absolute;
  left: 0;
  top: 0;
  width: 65vw;
  height: 10px;
  transform-origin: 0 50%;
  border-radius: 50%;
  opacity: 0.65;
  filter: blur(3px);
}

.ray1 {
  transform: rotate(0deg);
  background: linear-gradient(
    to right,
    rgba(255, 0, 100, 0.95),
    rgba(255, 0, 100, 0)
  );
}

若希望光束更多，可以繼續新增 ray13、 ray14 等元素，並調整旋轉角度。

6. 加入旋轉動畫

6.1 旋轉全部光束

@keyframes rotateRays {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}

.rays.active {
  animation: rotateRays 12s linear infinite;
}

6.2 旋轉 DISCO 球

@keyframes spinBall {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}

.disco-ball.on {
  animation: spinBall 5s linear infinite;
}

數字越小，旋轉速度越快。例如 5s 改成 2s， 球體就會轉得更快。

7. 使用 JavaScript 控制開關

JavaScript 不必直接修改大量 CSS，只要加入或移除類別即可。 

function turnOn() {
  discoBall.classList.add("on");
  rays.classList.add("active");
  lightCore.classList.add("active");
}

function turnOff() {
  discoBall.classList.remove("on");
  rays.classList.remove("active");
  lightCore.classList.remove("active");
}

classList.add() 是加入類別， classList.remove() 是移除類別。 這是動態網頁中非常常見的控制方式。

10. 常見問題與排除方式

問題 可能原因 解決方式
光線不像從球射出 光線和球體使用不同定位中心。 把光線與球放入同一個 disco-system 中。
光束從中間向兩邊延伸 旋轉原點仍在元素中央。 設定 transform-origin: 0 50%。
按鈕沒有反應 元素 ID 或函式名稱拼錯。 確認 id="rays" 與 JavaScript 中名稱完全一致。
球體擋不住光線 圖層順序錯誤。 讓球體的 z-index 高於光線。
手機畫面太大 沒有設定響應式樣式。 在 @media 中縮小球體並調整光束長度。

11. 可以繼續加入的功能

速度控制 增加滑桿，讓使用者調整球體與光線的旋轉速度。
色彩模式 加入彩虹、冷色、暖色或單色燈光切換。
音樂互動 搭配 Web Audio API，讓光束隨音量或節奏變化。
全螢幕模式 加入 Fullscreen API，作為舞台或教室展示效果。
隨機光束 透過 JavaScript 隨機改變光束角度、長度與亮度。
3D 球體 使用 CSS 3D 或 Three.js 製作更真實的立體鏡面球。
