一、文件概覽

這份技術文件的目標，不只是展示動畫成果，而是幫助教師或教材製作者理解：怎麼用 CSS 把自然科學概念轉成可視化動畫。以本案例來說，我們把水循環拆成幾個可觀察的動態歷程：

  • 太陽持續發光，代表熱能來源。
  • 海面波浪持續變動，讓畫面更有自然感。
  • 蒸發：水蒸氣由海面上升。
  • 凝結：雲逐漸形成並移動到山區。
  • 降雨：雨滴自然落下。
  • 地表逕流與河流：水回到海洋。
  • 地下滲透與地下水：用半透明藍色箭頭表示水進入土壤、再慢慢回流。

二、動畫範例一：完整循環播放版

這是 15 秒自動循環播放的版本，適合放在課堂導入、教學影片、簡報頁面或展示牆。學生可以先從整體觀察水如何在海洋、空氣、山區、地表與地下之間循環。

特色：整體流程清楚、畫面柔和、全程自動播放，適合作為課堂導入或整體概念說明。

觀課建議：可以請學生先說出他在畫面中看到了哪些自然元素，再由老師引導連結到「蒸發 → 凝結 → 降雨 → 地表逕流 → 地下滲透 → 回到海洋」的完整歷程。

三、動畫範例二：互動點選學習版

這個版本將流程拆成四個主要學習按鈕：蒸發、凝結、降雨、地下水。學生點擊不同階段後，畫面會強調該階段的動畫，右側同步顯示中文說明與科學解釋，適合分段教學與形成性提問。

特色：可分段聚焦、可重播、附中文說明與科學解釋，適合課堂討論與師生問答。

四、製作流程：從科學概念到動畫畫面

  1. 先定義教學概念。 把水循環拆成可教學、可觀察的步驟，而不是一開始就急著寫程式。這樣動畫才會對應教材重點。
  2. 規劃畫面元素。 本案例固定包含太陽、海洋、河流、山脈、雲、樹木與植被，讓學生能從真實自然景觀中理解水循環。
  3. 決定哪些元素需要動。 例如太陽發光、波浪搖動、水蒸氣上升、雲移動、雨滴下落、地下水箭頭流動等。
  4. 用圖層概念安排場景。 背景天空、山脈、地面、海洋、河流、植被、動態水汽與雲雨，各自分層，才容易控制前後關係。
  5. 用 @keyframes 製作動畫。 每一種自然現象設計一段動畫，再透過 animation 屬性讓它重複播放。
  6. 若要互動，就加入按鈕與說明面板。 利用資料屬性（例如 data-stage）切換狀態，再用 JavaScript 更新說明文字。

五、程式重點解析

1. 基本 HTML 場景結構

建立動畫時，先把場景元素放好，再讓它們動。下面是簡化過的結構範例：

<div class="sun"></div>
<div class="ocean">
  <div class="wave one"></div>
  <div class="wave two"></div>
  <div class="wave three"></div>
</div>
<div class="vapor-area">
  <div class="vapor v1"></div>
  <div class="vapor v2"></div>
</div>
<div class="cloud-system">
  <div class="cloud"></div>
</div>

2. 用 @keyframes 設計自然現象

CSS 動畫的核心，就是透過 @keyframes 描述「從哪裡變到哪裡」。以下是太陽發光、水蒸氣上升與雨滴落下的簡化範例：

@keyframes sunGlow {
  from { filter: brightness(1); transform: scale(1); }
  to   { filter: brightness(1.14); transform: scale(1.04); }
}

@keyframes vaporRise {
  0%   { opacity: 0; transform: translateY(28px) scale(.9); }
  20%  { opacity: .55; }
  100% { opacity: 0; transform: translateY(-110px) scale(1.25); }
}

@keyframes rainFall {
  0%   { opacity: 0; transform: translateY(-8px); }
  20%  { opacity: .95; }
  100% { opacity: 0; transform: translateY(150px); }
}

3. 互動版的按鈕結構

如果要讓學生點擊不同階段，就要先準備按鈕，並利用 data-stage 記錄每個階段名稱：

<nav class="controls">
  <button class="stage-btn active" data-stage="evaporation">蒸發</button>
  <button class="stage-btn" data-stage="condensation">凝結</button>
  <button class="stage-btn" data-stage="rain">降雨</button>
  <button class="stage-btn" data-stage="groundwater">地下水</button>
</nav>

4. 切換說明與動畫狀態

互動版利用 JavaScript 根據按鈕切換內容，並更新右側說明面板。下面是核心概念的簡化寫法：

const content = {
  evaporation: {
    tag: '階段 1',
    title: '蒸發 Evaporation',
    desc: '太陽加熱海面，水分子變成水蒸氣後上升。'
  },
  condensation: {
    tag: '階段 2',
    title: '凝結 Condensation',
    desc: '水蒸氣遇冷形成許多細小水滴，逐漸形成雲。'
  }
};

function setStage(stage) {
  slide.dataset.stage = stage;
  buttons.forEach(btn => btn.classList.toggle('active', btn.dataset.stage === stage));
  stageTitle.textContent = content[stage].title;
  stageDesc.innerHTML = content[stage].desc;
}

5. 本案例使用到的關鍵技術

技術用途在本案例的應用
linear-gradient / radial-gradient做自然色塊與光影天空、太陽、海洋、土地、雲朵明暗
clip-path裁切形狀山脈與河流輪廓
::before / ::after補充裝飾層太陽光暈、山頂明暗、河流流光
animation播放動畫波浪、蒸氣、雲移動、雨滴、地下水
data-stage切換學習階段互動版按鈕控制不同過程

六、教學實作建議

課堂流程建議

  1. 先播放完整循環版，請學生觀察畫面中有哪些景物。
  2. 請學生依序說出水可能經過哪些地方。
  3. 再打開互動版，逐一點選蒸發、凝結、降雨、地下水。
  4. 每個階段都先讓學生預測，再觀看動畫，再回答科學解釋。
  5. 最後請學生畫出自己的水循環圖，或口頭解說完整流程。

延伸任務

  • 把「地表逕流」做成可點擊的第五個按鈕。
  • 加入「蒸散作用」說明植物也會把水分釋放到空氣中。
  • 讓學生自行改變動畫顏色，做晴天與雨天版本。
  • 挑戰製作「聖嬰現象」或「河川侵蝕」等其他自然主題動畫。
如果要給學生操作，建議讓按鈕區更大、文字更少一點；如果要給老師示範，則可保留完整說明與英文專有名詞。
