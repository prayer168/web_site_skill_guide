一、什麼是 Zeabur AI Hub？

Anthropic API 代理閘道（Gateway）

§ 01
特點 說明
按用量付費 用多少付多少，不使用當月費用為 $0
無需訂閱 不需 Claude Pro $20／月的固定費用
支援多模型 Claude、Gemini、GPT 等主流模型均可存取
亞洲節點 東京節點 (hnd1) 提供台灣／亞洲低延遲連線

二、費用比較

月費訂閱 vs. 按用量計費

§ 02
方式 費用 適合對象
Anthropic 官方訂閱 $20 / 月（固定） 每天大量使用的重度用戶
Zeabur AI Hub 依 token 用量計費 偶爾使用、想控制成本的開發者

週末小專案（約 2 小時使用） ~ $0.50 美元

每日密集開發（等同 Pro 用量） 可能超過 $20，建議直接訂閱 Pro

三、前置需求

安裝 CLI 並取得 API Key

§ 03
3.1
安裝 Claude Code CLI

確認已安裝 Node.js 18 以上版本，然後執行：

terminal 
npm install -g @anthropic-ai/claude-code
claude --version
3.2
取得 Zeabur AI Hub API Key
步驟 操作說明
1 前往 zeabur.com 並登入帳號
2 點選 Dashboard > AI Hub
3 建立或複製你的 API Key

五、設定方式

臨時使用或永久寫入 shell 設定檔

§ 05
A
方案 A：臨時使用（單次會話）

只要在終端機執行以下指令，本次會話即套用 Zeabur 設定：

Windows CMD 
# Windows CMD
set ANTHROPIC_BASE_URL=https://hnd1.aihub.zeabur.ai/
set ANTHROPIC_AUTH_TOKEN=你的_Zeabur_API_Key
set CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1
macOS / Linux / WSL 
# macOS / Linux / WSL
export ANTHROPIC_BASE_URL="https://hnd1.aihub.zeabur.ai/"
export ANTHROPIC_AUTH_TOKEN="你的_Zeabur_API_Key"
export CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1
B
方案 B：永久設定（加入 shell 設定檔）

讓每次開啟終端機都自動套用設定：

bash / zsh 
# 步驟 1：開啟設定檔
nano ~/.zshrc          # zsh（macOS 預設）
# 或
nano ~/.bashrc         # bash（Linux / WSL）

# 步驟 2：在檔案最底部加入以下三行
export ANTHROPIC_BASE_URL="https://hnd1.aihub.zeabur.ai/"
export ANTHROPIC_AUTH_TOKEN="你的_Zeabur_API_Key"
export CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1

# 步驟 3：儲存後重新載入（Ctrl+O → Enter，Ctrl+X 退出）
source ~/.zshrc
⚠️
必須設定 CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1，否則 Claude Code 會送出實驗性標頭，導致 Zeabur 閘道回傳 invalid beta flag 錯誤。
🚀

六、啟動與使用

啟動 Claude Code 並驗證連線

§ 06

啟動 Claude Code：

terminal 
# 使用 Sonnet 4.5 模型（推薦）
claude --model claude-sonnet-4-5

# 或直接啟動（使用預設模型）
claude

驗證連線是否正常：

terminal 
# 確認環境變數已設定
echo $ANTHROPIC_BASE_URL

# 啟動後詢問一個簡單問題確認連線
claude
> 你好，請說一句話確認連線正常
🔧

七、常見問題排解

錯誤訊息與對應解法

§ 07
問題 原因 解決方式
invalid beta flag 錯誤 未設定停用 beta 標頭 確認 CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1 已設定
連線逾時 / 速度慢 節點選擇不佳 台灣改用 hnd1（東京）節點
401 Unauthorized API Key 錯誤或過期 至 Zeabur Dashboard 重新產生 API Key
claude 找不到指令 PATH 未設定正確 確認 npm 全域路徑已加入 PATH 環境變數
環境變數重開後消失 只有臨時設定 改用方案 B，將設定寫入 ~/.zshrc 或 ~/.bashrc
📋

八、設定快速摘要

三個環境變數一次搞定

環境變數 設定值 用途
ANTHROPIC_BASE_URL https://hnd1.aihub.zeabur.ai/ 指向 Zeabur 代理伺服器
ANTHROPIC_AUTH_TOKEN 你的 Zeabur API Key 認證身份
CLAUDE_CODE_DISABLE_
EXPERIMENTAL_BETAS		 1 避免 beta 標頭錯誤
💡
完成設定後，Claude Code 的所有功能與命令完全相同，只是計費方式從月費改為按用量。
